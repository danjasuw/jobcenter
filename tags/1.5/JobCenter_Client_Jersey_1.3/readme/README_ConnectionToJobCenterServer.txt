

The connection to the server must be set by copying the file

"jobcenter_client_code/client_server_interface_jersey_1.3/config_sample_files/config_server_connection.properties"

to the directory

"jobcenter_client_code/client_server_interface_jersey_1.3/config"

Then configure the connection URL to the JobCenter Server in that copy of the file in the "config" directory.
